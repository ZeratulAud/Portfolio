My Projects: AIMS

For my Senior Project we are updating and adapting a codebase given to us by Dr. Alireza Tavakkoli. The Project contains all the necessary components to perform teleoperation on remote robots by utilizing VR controls and game engine renders. The project was built with Unreal Engine 4 and uses an Oculus Rift to explore the enviroment and give commands to the connected robot. We were tasked with updating the codebase to work with a new robot with new sensors, two new physical locations, updated software, and new objects that we will manually model into digital to-scale assets.

My Projects: 3D Donkey Kong

Previous Next

In my Computer Graphics class we recreated Donkey Kong using OpenGL and bullet physics. In this project we had to use assimp to load in models we found on the internet as well as models that we created ourselves. Also you may be able to see the specular hilights that show up on the image of donkey kong which were rendered by using a phong fragment shader. In this class we also created a solar system simulation and a physics based pinball game.

My Projects: Naval Invasion

This project consisted of building a game engine from scratch using the Ogre 3D graphics engine. In this project my team and I implemented various ships, firing patterns and, AI behaviors to allow for fleet movement as well enemy patrol and pathing. From this project I was able to learn a lot about proper engine architechure using Object-Oriented Programming and many cool AI techniques from my professor who does research in Artificial Intelegence at the University of Nevada Reno.
