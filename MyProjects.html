AIMS

For my Senior Project we are updating a nolonger functioning codebase given to us by Dr. Alireza Tavakkoli. This codebase is called the VETO Codebase. This contains all the necessary components to perform teleoperation on remote robots by utilizing VR controls and game engine renders. The codebase does its rendering with Unreal Engine 4 and operates control from within an Oculus Rift environment. Our project involves us updating the codebase to work with a robot with new sensors, two physical locations, updated software, and new objects that we will manually model into digital to-scale objects.

In this project my group of 3 recreated Donkey Kong using OpenGL and bullet physics. We also had to use assimp to load in models we found on the internet as well as models that we created ourselves. I took lead on the shader programing which is shown by the specular highlights in the image of donkey kong or the lighting from the other project images. This was done using a Blinn-Phong shading model to render the lighting of the sene efficiently with control over various lighting values.

Naval Invasion

This project consisted of building a game engine from scratch using the Ogre 3D graphics engine. In this project my team and I implemented various ships, firing patterns, and AI behaviors to allow for fleet movement as well enemy patrol and pathing. My main focus was the player controller along with fleet mechanics to have your ally ships follow your main ship in different patterns and having them assist your main ship when attacking. From this project I was able to learn a lot about proper engine architecture using Object-Oriented Programming and AI techniques from a research professor who specializes in Artificial Intelligence at the University of Nevada Reno.
